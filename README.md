# Nimbus Landing Page

Marketing site for [Nimbus Cloud](https://nimbus-example.dev).

## Development
fdsqfdsqdsq
```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

Output is written to `dist/`.

## Deployment

Staging deploys run automatically every 4 hours via GitHub Actions. Production deploys are triggered manually through the deployment dashboard.
